    Introduction

    Welcome to the ConfidentialCloud API!

    You can use our API to access all public ConfidentialCloud API endpoints.

    Authentication

    curl "https://api.confidentialcloud.io/v1" \
  -H "Authorization: <your_api_key>"

    ConfidentialCloud uses API keys to allow access to the API. You can register a new API key at our developer portal.

    ConfidentialCloud expects for the API key to be included in all API requests to the server in a header that looks like the following:

    Authorization: <your_api_key>

    You must replace `` with your personal API key.

    Enclaves

    Get all Enclaves

    curl "https://api.confidentialcloud.io/v1/enclaves"

    This endpoint retrieves all Enclaves.

    HTTP Request

    GET https://api.confidentialcloud.io/v1/enclaves

    Get a specific Enclave

    curl "https://api.confidentialcloud.io/v1/enclave/2"

    This endpoint retrieves a specific kitten.

    HTTP Request

    GET https://api.confidentialcloud.io/v1/enclave/<ID>

    URL Parameters

    Parameter Description
    ID The ID of the Enclave to retrieve

    Delete a specific Enclave

    curl "https://api.confidentialcloud.io/v1/enclave/2" \
  -X DELETE

    This endpoint deletes a specific Enclave

    HTTP Request

    DELETE http://example.com/kittens/<ID>

    URL Parameters

    Parameter Description
    ID The ID of the kitten to delete
